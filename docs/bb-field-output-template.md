# Modèle de fiche de sortie – Version terrain

Chaque dégradation identifiée doit générer une fiche standardisée.

---

## Exemple – Nid de poule (56)

Type ME52 :
Nid de poule (56)

Description courte :
Cavité localisée résultant d’une désagrégation complète.

Niveau d’urgence :
ÉLEVÉ – Intervention rapide nécessaire

Cause dominante probable :
- Infiltration d’eau
- Faïençage évolué
- Défaut structurel localisé

Action immédiate recommandée :
- Bouchage temporaire (à froid ou blow patcher)
- Sécurisation si trafic important

Remédiation durable :
- Découpe propre
- Reprise localisée en enrobé chaud
- Vérification portance support

Risque si non traité :
- Agrandissement rapide
- Danger usagers
- Responsabilité gestionnaire

---

## Format de sortie simplifié mobile

[ TYPE ]
[ URGENCE ]
[ ACTION IMMÉDIATE ]
[ REMÉDIATION DURABLE ]

# Extension future – Module Pro (non activé en V1)

Objectif :
Permettre l’ajout de recommandations techniques avancées
selon les équipements disponibles ou la stratégie d’entretien.

Exemples de modules activables :

- Réparation projetée sous pression
- Enrobé chaud traditionnel
- Micro-enduit
- Traitement préventif
- Planification budgétaire

Ce module sera désactivé dans la version publique V1.
